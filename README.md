# 美食大战老鼠同人游戏

## 软件简介

《美食大战老鼠：重生》是使用 GameMaker 引擎重新编写的美食大战老鼠同人游戏，旨在回归塔防初衷，优化不合理的游戏体验。

## 分支说明

本仓库包含以下主要分支：

- **master**：稳定版分支，用于发布经过测试的稳定版本。
- **develop**：开发版分支，用于进行日常开发和功能迭代。
- **community**：社区贡献分支，无保护，用于接受社区提交的代码和贡献。
